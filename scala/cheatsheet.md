
# Value types

```scala 

var i = 12 // Mutable
val j = 10 // Immutable

// Boolean
b: Boolean = false
byte: Byte = 1

// Integers
i: Short = 2 // 16 bits
j: Int = 2 // 32 bits
k: Long = 2 // 64 bits

// Floating points
f: Float = 2.4 // 32 bits
d: Double = 45.3 // 64 bits

// chars and String
val s: String = "Hello, World"
val c: Char = 'H'
```
