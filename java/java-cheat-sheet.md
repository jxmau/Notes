<h2> Primitive types </h2>

| name | type | size | default |
| --- | --- | --- | --- |
| byte |  | 8 bits | 0
| short | Integer number  | 16 bits | 0
| int | Integer number  | 32 bits | 0
| long | Integer number  | 64 bits | 0L
| float | floating point | 32 bits | 0.0f
| double | floating point | 64 bits | 0.0d
| char | character | 16 bit |
| boolean | Boolean value | 8 bit (?) | false

<h2> Operator </h2>

| Name | Symbol |
| --- | --- |
| Equal | == or .equals() |
| Different | != or !foo.equals(bar) |
| More | < |
| More | > |

<h2> Get the size</h2>

| fun/meth | for |
| --- | --- |
| .length | Arrays |
| .length() | String and String related object (StringBuilder, etc.) |
| .size() | Collections |
